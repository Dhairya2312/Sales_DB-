# Retail Sales Analytics — SQL Server Project

## Project Overview

This project is a Retail Sales Analytics system built using SQL Server.  
The main objective is to analyze sales data to understand customer behavior, product performance, and revenue trends in a retail business.

Through this project, raw transactional data is converted into meaningful insights using SQL queries, helping simulate real-world business decision-making.

---

## Tools & Technologies

- Microsoft SQL Server  
- SQL Server Management Studio (SSMS)

---

## Database Design

**Database Name:** RetailSalesDB  

### Tables Included

| Table Name | Description |
|-----------|------------|
| Customers | Stores customer information such as name, city, and age |
| Products  | Contains product details like name, category, and price |
| Orders    | Records all transactions linking customers with products |

---

## Concepts Applied

This project demonstrates the use of key SQL concepts:

- DDL (Data Definition Language): CREATE TABLE, PRIMARY KEY, FOREIGN KEY  
- DML (Data Manipulation Language): INSERT, SELECT  
- Joins: INNER JOIN across multiple tables  
- Aggregate Functions: SUM, AVG, COUNT  
- GROUP BY and HAVING clauses  
- Subqueries  
- Date Functions: MONTH(), YEAR(), DATEADD(), GETDATE()  
- Filtering and Sorting: WHERE, ORDER BY, TOP N

## Analytics Queries & Insights

The following queries were created to answer practical business questions:

| Query No. | Business Insight |
|----------|----------------|
| 1 | Total revenue generated by each product |
| 2 | Top 3 customers based on total spending |
| 3 | Monthly sales trend throughout the year |
| 4 | Revenue breakdown by product category |
| 5 | Customers who have never placed an order |
| 6 | Most frequently ordered product |
| 7 | Average order value per customer |
| 8 | Revenue comparison by city |
| 9 | Orders placed in the last 3 months |
| 10 | Products generating revenue above ₹20,000 |

---

## Key Findings

- The Electronics category generated the highest overall revenue  
- Laptops and Mobile phones are the top-performing products  
- Ahmedabad and Surat are the leading cities in terms of revenue  
- A small group of customers contributes a major portion of total sales
